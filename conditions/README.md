# conditions/ - シミュレーション条件（入力）

シミュレーション条件ファイルを格納するディレクトリ。

## ディレクトリ構成

```
conditions/
└── {cond_id}/                    # 条件ID（例: 5_08_mod_logging2）
    ├── sim_conds.xlsx            # シミュレーション条件
    └── operation_schedule.xlsx   # 稼働工程表（省略可）
```

## sim_conds.xlsx - シミュレーション条件

各塔の設定を含むExcelファイル。

**主なシート:**
- 共通設定: セクション数、計算ステップ時間
- 充填層条件: 吸着材密度、空隙率、粒子径
- 容器条件: 直径、高さ、壁厚
- 供給ガス条件: 流量、組成、温度
- ストリーム条件: 断面積、吸着材量

## operation_schedule.xlsx - 稼働工程表

各工程で各塔が実行する運転モードと終了条件を定義。

**列構成:**
| 塔1 | 塔2 | 塔3 | 終了条件 |
|-----|-----|-----|---------|
| 停止 | 停止 | 停止 | 時間経過_0.4_min |
| 初回ガス導入 | 停止 | 停止 | 圧力到達_塔1_0.1_MPaA |
| ... | ... | ... | ... |
