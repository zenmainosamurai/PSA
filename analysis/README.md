## supp_symbolic_regression.ipynb

実験データから吸着平衡式（吸着等温線）を導出するノートブック。

**導出された式:**
```
q = P * (a1 - a2*T) / (P - b1*(1 - b2*sqrt(T))^3 + b3)
```

**係数:**
- a1 = 252.0724
- a2 = 0.50989705
- b1 = 3554.54819062669
- b2 = 0.0655247236249063
- b3 = 1.7354268

**単位:**
- P: 圧力 [kPaA]
- T: 温度 [K]
- q: 平衡吸着量 [cm³/g-abs]

この式は `physics/adsorption_isotherm.py` で使用されている。
